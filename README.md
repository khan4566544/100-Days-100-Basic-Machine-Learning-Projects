**100 Days 100 Basic Machine Learning Models**

Welcome to my 100-day machine learning challenge!
This repository contains 100 small, beginner-friendly ML projects, each completed in a single day.
The purpose is simple: learn, improve, and stay consistent.

**Overview**

Every day features:
One dataset
One ML algorithm
Clean, simple code
A short explanation
Results + visuals
A self-contained project folder

No overkill. Just pure learning.


**Libraries**
Python
NumPy / pandas
scikit-learn
Matplotlib / Seaborn
Jupyter Notebook



**Goals**

Build strong fundamentals
Practice daily
Create a large ML portfolio
Explore many algorithms quickly



**Progress**
Updating daily until Day 100!
